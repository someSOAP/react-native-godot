# React Native Godot (Fork)

> [!IMPORTANT]
> This fork of the library is powered by [react-native-worklets](https://docs.swmansion.com/react-native-worklets/) (from Software Mansion). It utilizes worklets to ensure high-performance communication between the React Native JavaScript thread and the Godot Engine thread.

This is a fork of the original repository: [borndotcom/react-native-godot](https://github.com/borndotcom/react-native-godot)
